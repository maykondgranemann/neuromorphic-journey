# 01. The Atom: LIF Neuron Simulation

This milestone focuses on modeling the **Leaky Integrate-and-Fire (LIF)** neuron, the fundamental unit of spiking neural networks.

## Objectives
- Understand the biological inspiration behind the LIF model.
- Simulate membrane potential decay and spike generation using **sPyNNaker**.
- Visualize the "Sawtooth" firing pattern.

## Technical Setup
- **Framework:** sPyNNaker (University of Manchester)
- **Environment:** Python 3.10+ (Ubuntu)
